Detailed Description

Our Player type. This type handles user input or computer move computation. It also stores the player name and the last move done, in order to save computation when checking the state of the game.

The Player.depth parameter will only be used if Player.is_computer is true. It will specify the depth of the AI (minimax) algorithm.
